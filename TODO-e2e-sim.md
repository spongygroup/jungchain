# E2E 시뮬레이션 수정사항

## 확인된 이슈
1. **23개 TZ (24개 X)** — 시작자 TZ 제외, 나머지 23개만 릴레이
2. **유저 사전배치 X → 넘어갈 때 매핑** — 미리 72명 세팅하지 말고, 해당 TZ 차례 올 때 유저 풀에서 매핑
3. **같은 TZ에 여러 도시** — "UTC+1" 대신 실제 도시명 표시, 또는 "다음 2시로 넘어가는 중입니다" 식으로 시간 기준 안내
4. **각 유저별 텔레그램 언어로 진행/안내** — 유저의 `language_code` 기반으로 정지기 코멘트, UI 메시지 전부 해당 언어
