# 정(情) — 24시간 감정 릴레이

## 정(情)이란?

매 정시, 한 사람이 보낸 사진 한 장과 짧은 글이 해가 지는 방향(서쪽)으로 다음 타임존의 누군가에게 전달됩니다. 받은 사람은 1시간 안에 자신만의 사진과 글로 정을 이어갑니다. 24시간, 24개 타임존을 거쳐 지구를 한 바퀴 돌아 원래 보낸 사람에게 돌아옵니다.

## 핵심 특징

- **사진 1장 + 텍스트(200자)** — 가볍지만 진심을 담을 수 있는 단위
- **정지기(AI 모더레이터)** — 모든 콘텐츠를 검증하고 따뜻한 코멘트를 남깁니다
- **포크** — 하나의 정이 여러 갈래로 나뉘어 나무처럼 퍼져나갑니다
- **온체인 기록** — 완주된 정은 Base L2에 영구 기록됩니다
- **17개 언어 지원** — 각 타임존의 현지 언어로 자동 번역

## 포크 시스템

같은 타임존에 여러 사람이 있으면, 첫 번째 사람은 원래 체인에 기록하고 나머지는 각각 새로운 갈래(포크)를 만듭니다. 포크는 부모 체인의 기존 블록을 복사한 뒤(Copy-on-Fork) 자기만의 길을 이어갑니다. 모든 포크는 원래 체인의 시작 시간 기준 24시간에 함께 종료됩니다.

```
Root Chain #10
  Seoul → Singapore → Bangkok → ... → Sydney ✅

  Fork #11 (Bangkok에서 분기)
  Seoul → Singapore → [Bangkok 다른 사람] → ... → Sydney ✅

  Fork #12 (Bangkok에서 또 분기)
  Seoul → Singapore → [Bangkok 또 다른 사람] → ... → Sydney ✅
```

## HAB (Human + AI + Blockchain)

정은 합(HAB) 프로토콜의 첫 번째 체인입니다.

- **정체인** — 합의 첫 번째 체인 (감정·관계)
- **일체인** — 합의 두 번째 체인 (노동·협업)
- **장체인** — 합의 세 번째 체인 (거래·커머스)

Human이 행동하고, AI가 돕고, Blockchain이 기록합니다.
코드는 복제할 수 있지만, 사람들이 만든 신뢰는 복제할 수 없습니다.
