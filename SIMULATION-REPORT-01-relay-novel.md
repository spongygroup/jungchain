## 결과

### 스토리 요약
오클랜드 새벽, 한 남자가 사진 삭제를 강요하며 시작. 조개껍데기 목걸이를 단서로 시드니 → 서울(열차에서 핏자국 발견) → 방콕(불교 부적) → 다카(종이 쪽지) → 카라치(엠프레스 마켓 접선) → 두바이(금 시장) → 모스크바(아파트 열쇠) → 카이로(파피루스) → 파리(나비고 카드) → 런던(추적) → 채링크로스 폐플랫폼(미국인 요원) → 상파울루(거짓 기억 각성) → 뉴욕(또 다른 거짓) → 시카고(새 정보원) → 덴버(강제 경로 변경) → LA(가짜 해독) → 알래스카(감금) → 하와이(대면) → 사모아에서 "나는 아들이었다"로 결말.

## 문제점

### 🔴 크리티컬
1. **덴버/LA 내용 중복** — Gemini가 `***` 구분선 후 같은 장면을 다른 버전으로 다시 생성. 동일 내용이 2배로 출력됨
2. **장르 이탈** — 로맨스로 지정했으나 완전 스파이 스릴러로 전개. Auckland 프롬프트에 "Romance/thriller" 병기가 원인

### 🟡 언어 문제
3. **정지기 3곳 한국어 출력** — 솔로몬 제도, 아조레스, 알래스카. 프롬프트에 언어 지정 없음
4. **일부 유저 언어 미준수** — João(PT→EN), Maria(ES→EN), Sofia(ES→EN). 프롬프트만으로 불충분

### 🟡 내용 문제
5. **패턴 반복** — "인물 등장 → 물건 전달 → 다음 도시" 10회 이상 반복
6. **주인공 수동적** — 계속 끌려다님. 자기 의지 행동 거의 없음
7. **등장인물 일회용** — 전부 나타났다 사라짐. 재등장/감정이입 없음
8. **선택지 무의미** — A든 B든 결국 "다음 도시 이동"이라는 같은 결과
9. **설정 모순** — 일반인→갑자기 요원, "폰타 델가다"는 이름이라더니 다음 블록에서 무시, 애스펀이라더니 알래스카
10. **결말 복선 없음** — "사모아 출신 아들"이라는 결말에 24블록 내내 아무 힌트 없음

### 🟡 UX 문제
11. **타임존-캐릭터 불일치** — UTC-4(뉴욕)에 카라카스 유저, UTC-6(덴버)에 멕시코시티 유저. 소설 내용은 미국 배경인데 캐릭터는 라틴아메리카

## 인사이트 — 실제 유저가 돌린다면

### 참여 허들
1. **150~300자 + 선택지 만들기는 부담이 크다** — 실제 유저는 작가가 아님. "한 줄만 써도 OK" 같은 가벼운 옵션 필요
2. **이전 내용을 번역해서 보여줘도 길다** — 3블록만 지나도 번역본이 수백 자. 읽기 귀찮으면 이탈. 요약본(3줄) + 전문(접기) 구조가 나을 수 있음
3. **5분 타임아웃은 짧을 수도** — 바쁜 사람은 알림 받고 5분 안에 소설을 못 씀. 1시간 or "다음 정각까지" 같은 여유가 현실적
4. **위치 공유 허들** — 프라이버시 민감한 유저는 거부할 수 있음. 위치 대신 타임존 직접 선택 옵션 필요

### 재미 요소
5. **선택 게임은 확실히 작동한다** — 이전 "따뜻한 메시지 릴레이"보다 참여 동기가 명확함. "내가 고른 선택이 스토리를 바꾼다"는 강력한 훅
6. **근데 선택이 실제로 결과를 바꿔야 한다** — 이번 시뮬에서 A든 B든 같은 결과로 수렴. 유저가 이걸 알아채면 "내 선택이 의미 없네" → 이탈
7. **장르 투표가 있으면 좋겠다** — 체인 시작 전에 "오늘의 장르: 로맨스/스릴러/SF/코미디" 투표. 유저가 장르를 고르는 것 자체가 참여
8. **내 파트가 전체에서 어디쯤인지** — "너는 24명 중 9번째야" 같은 위치감. 지구본 위에 내 위치가 점으로 찍히는 느낌

### 리텐션
9. **기다림이 핵심인데, 너무 길면 잊는다** — 실제 서비스에선 1타임존=1시간. Seoul 유저는 체인 시작 후 ~9시간 뒤에 차례. 9시간을 기다리게 하려면 중간 리포트가 핵심
10. **완주 후 결과물이 "내 작품"이 되어야 한다** — 전체 소설을 예쁘게 포맷해서 공유 가능한 카드/이미지로 만들면 바이럴 가능성 ↑
11. **연속 참여 동기** — "어제 네가 쓴 체인이 완주됐어! 오늘도 새 체인이 시작됐어" 같은 알림. 매일 다른 장르/테마
12. **내가 쓴 부분 하이라이트** — 완주 결과에서 "🟡 여기가 네가 쓴 부분" 표시. 24명 중 내 기여가 보여야 뿌듯함

### 콜드 스타트
13. **24명이 모여야 체인이 돌아감** — 초기엔 AI가 대부분 채움. 근데 "AI 19명 + 사람 5명"이면 유저 입장에서 "이거 나 혼자 하는 거 아냐?" 느낌
14. **작은 체인부터 시작** — 6타임존(4시간 간격), 12타임존(2시간 간격) 같은 미니 체인으로 시작. 유저가 늘면 24로 확장
15. **AI 참여를 투명하게** — "이 블록은 정지기(AI)가 이어썼어"를 보여줘야 신뢰. 숨기면 나중에 "다 AI였네" 반감

### 수익화 연결
16. **완주 체인 NFT/수집** — 24블록 완주된 체인을 디지털 수집품으로 만들면 참여 동기 + 수익
17. **프리미엄 장르/테마** — 기본은 무료, 특별 테마("크리스마스 릴레이", "공포의 밤") 는 프리미엄
18. **작가 랭킹** — 내가 참여한 체인 중 가장 많이 읽힌/공유된 체인. 경쟁 요소
